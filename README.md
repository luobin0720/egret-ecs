# egret-ecs
基于Egret开发的ECS框架，使之更加易用.

> 该项目还在施工中! 欢迎参与进行改进

## 项目框架

- src
  - core 核心文件
  - debug 调试文件
  - ecs ECS框架
    - components 组件
      - renderables 渲染组件
    - internalUtils 内部帮助类
    - systems 系统类
